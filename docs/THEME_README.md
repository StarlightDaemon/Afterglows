# Theme Files

This directory contains the theme system pulled from the main StarlightDaemon portfolio site.

## Files

- **styles.css** - Main stylesheet with CRT terminal theme
- **theme-toggle.css** - Settings panel styling
- **script.js** - Core scripts
- **theme-toggle.js** - Theme switching logic (P1 Green, P3 Amber, P4 White, IBM Blue, Matrix)

## Themes Available

The site supports multiple retro terminal color schemes:
- **P1 GREEN** (Default) - Classic green phosphor terminal
- **P1+ MATRIX** - Enhanced Matrix green with scanlines
- **P3 AMBER** - Warm amber CRT
- **P4 WHITE** - High contrast white phosphor
- **IBM BLUE** - IBM terminal blue

## Fonts Available

- **IBM Plex Mono** (Default)
- **VT323** - Old school terminal
- **Space Mono**
- **Fira Code**
